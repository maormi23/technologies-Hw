א. שפות צד לקוח

1. HTML – משמשת לבניית מבנה הדף כמו כותרות פסקאות וטפסים

2. CSS – אחראית על עיצוב הדף צבעים גופנים ומיקום אלמנטים

3. JavaScript – מאפשרת אינטראקטיביות כמו כפתורים טפסים ושינויים בזמן אמת

ב. ענן מחשוב ויתרונות

ענן מחשוב הוא שירות אינטרנטי שמאפשר לאחסן ולהפעיל תוכנות ומידע מרחוק בלי צורך בשרת מקומי

יתרונות לעסקים קטנים:

  • חיסכון בעלויות רכישת חומרה ותשתיות
  • גישה למידע ולתוכנות מכל מקום ובכל זמן

ג. רשת תקשורת וסוגי רשתות

רשת תקשורת היא קישור של מחשבים ומכשירים שמאפשר העברת מידע ביניהם

סוגי רשתות נפוצות:

  • LAN – רשת מקומית בתוך בית או משרד
  • WAN – רשת רחבה שמחברת מיקומים שונים
